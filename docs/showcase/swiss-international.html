Introduction

Nexus CLI is a command-line tool for deploying, managing, and scaling your applications on the Nexus platform. It provides a streamlined workflow for developers who prefer working in the terminal.

New in v2.4

Preview deployments are now available on all plans. Use nexus deploy --preview to create shareable preview URLs for any branch.

Installation

Install the Nexus CLI globally via npm, pnpm, or Homebrew. The CLI requires Node.js 18 or later.

bashCopy
# npm
npm install -g @nexus/cli

# pnpm
pnpm add -g @nexus/cli

# Homebrew
brew install nexus-cli

Verify the installation:

bash
nexus --version
# → nexus/2.4.1 linux-x64 node-v22.0.0

Quick Start

Deploy your first application in under 2 minutes:

1

Authenticate

Log in with your Nexus account or create one if you don't have one.

nexus login
2

Initialize your project

Run nexus init in your project directory to link it to Nexus.

cd my-project
nexus init
3

Deploy

Push your code to production with a single command.

nexus deploy
# → Deploying to production...
# → ✓ Build complete (24s)
# → ✓ Live at https://my-project.nexus.app

Command Overview

CommandDescription
nexus initInitialize a new Nexus project in the current directory
nexus deployDeploy the current project to production
nexus deploy --previewCreate a preview deployment on a temporary URL
nexus statusShow the current deployment status
nexus logsStream live logs from the current deployment
nexus rollbackRoll back to a previous deployment
nexus env add KEY=valueAdd an environment variable
