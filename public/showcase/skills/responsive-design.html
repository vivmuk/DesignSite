Mobile-First vs Desktop-First

Two strategies for building responsive layouts. Mobile-first is the modern standard, starting with core content and progressively enhancing for larger screens.

Recommended

Mobile-First Approach

Start with the smallest screen and progressively enhance. Uses min-width media queries to add complexity as the viewport grows.

.container {
  width: 100%;
  padding: 1rem;
}

@media (min-width: 768px) {
  .container {
    max-width: 1200px;
    margin: 0 auto;
  }
}
Legacy

Desktop-First Approach

Start with the full desktop experience and use max-width media queries to subtract or hide elements for smaller screens.

.container {
  max-width: 1200px;
  margin: 0 auto;
}

@media (max-width: 768px) {
  .container {
    padding: 1rem;
  }
}
