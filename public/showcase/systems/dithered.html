// How it works

Ordered Dithering

Classic Bayer matrix patterns that produce clean, organized dot distributions. Perfect for UI elements that need consistent texture at any scale.

Error Diffusion

Floyd-Steinberg and Jarvis-Judice-Ninke algorithms that redistribute quantization error across neighboring pixels for organic, natural patterns.

Custom Patterns

Define your own threshold matrices for unique halftone effects. SVG-compatible and build-time ready for any rendering pipeline.

Multi-bit Output

Support for 1-bit through 8-bit output depths. Control exactly how many tonal levels your system supports at render time.
