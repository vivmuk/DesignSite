NextUI is a React UI library that helps you build beautiful and accessible user interfaces. Created on top of Tailwind CSS and React Aria.

Why NextUI?

Everything you need to build modern applications

Beautiful and Unique

Each component is crafted with attention to detail, providing a stunning default look.

Accessible

Built on top of React Aria, ensuring all components meet accessibility standards.

TypeScript Ready

Fully typed components and hooks for a better developer experience.

Customization

Powerful theming system with CSS variables and Tailwind CSS integration.

Dark Mode

First-class dark mode support with automatic switching.

Fast Performance

Optimized for speed with minimal re-renders and efficient bundle size.
