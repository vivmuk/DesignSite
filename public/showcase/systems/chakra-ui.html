Why Chakra UI?

Developer-friendly features that make building accessible React apps a breeze.

Ease of Styling

Chakra UI contains a set of layout components like Box and Stack that make it easy to style your components by passing props.

Flexible & Composable

Chakra UI components are built on top of React UI primitives for endless composability.

Accessible

Chakra UI follows WAI-ARIA guidelines specifications and has the right aria-* attributes for accessibility.

Dark Mode

Most components are compatible with dark mode. Just add the useColorMode hook and toggle away.

Themeable

Customize any part of the components to match your design. Create your own design system with ease.

Active Community

Join over 20k developers building with Chakra UI. Get help, share feedback, and contribute.
